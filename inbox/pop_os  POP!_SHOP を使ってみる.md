# pop_os  POP!_SHOP を使ってみる

POP!_SHOP で Blender と VScode と MEGAsync を入れた。

もう一度インストール済みを見るとアイコンが❌マークになってた。

```shell
$ flatpak update
$ flatpak uninstall -unused
$ flatpak repair --user
```

アイコンが出るようになった。
てゆーかここは時間かかってるだけっぽい。
ほっとけば出る。

MEGAsync は正規サイトからダウンロードした。
