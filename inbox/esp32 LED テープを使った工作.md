# LED テープ

シリアルLEDテープのスペック

-   駆動電圧 5V
-   消費電流　約55mA / 1 LED
-   RGB256色
-   PWM制御
-   3端子( 5V / SIG / GND )


1個55mA も流れる。外からの電源が必要かな。
