# A4 の pixel
2480 x 3508 pixel

A4  はインチで  w: 8.27 inch    h: 11.69 inch

300dpi だと
- w: 8.27 × 300 = 2480 pixel
- h: 11.69 × 300 = 3508 pixel

72dpi では
- w: 8.27 × 72 = 595 pixel
- h: 11.69 × 72 = 842 pixel

350dpi だと
- w: 8.27 × 350 = 2894.5 pixel = 2985 pixel
- h: 11.69 × 350 = 4091.5 pixel = 4092 pixel
